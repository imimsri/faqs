IT contractor guidelines

States external contractor technical specifications

External Computer Applications Requirements All external contractor applications should be able to run under a Linux based Kubernetes cluster. Contractor must be able to setup and update an inhouse git repository https://git.imim.cat with all needed Kubernetes deployment files. Git users will be allocated to acredited contractor staff members. Contractor must supply CI (continuous integration tests) to validate application releases IMIM will assign contractor a Kubernetes namespace per aplication. Application updates will be carried out through CI (continuous integration) workflow.
